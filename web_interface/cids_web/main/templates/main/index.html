CIDS

CIDS - це система відслідковування подій (аномальної поведінки та можливих атак) у хмарному сховищі підприємства. Завдяки зручному відображенню статистичних даних та попереджень, цей застосунок допомагає оперативно виявляти підозрілу активність у AWS Cloud S3 сховищах.

Dashboard - це основний розділ, що надає наочну агреговану інформацію про події у хмарі. Тут знаходиться діаграма частоти запитів, останні логи, а також дві вибірки: топ GET-запитів за IP-адресами та попередження, про ймовірні атаки.

Logs List містить повну інформацію про всі логи подій. Дані подані у вигляді таблиці, яку можна відсортувати або фільтрувати по ключовому слову.

Alerts - це сторінка з попередженнями, які було надіслано користувачу через Telegram Bot. Усі попередження містять час, повідомлення та тригер, завдяки чому можна легко виявити та попередити атаки на хмарне сховище.
